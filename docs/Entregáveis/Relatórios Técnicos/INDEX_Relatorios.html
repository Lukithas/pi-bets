📊 Relatório Técnico 01
Título: Análise Geral de Dados e Pesquisa
Arquivo: Relatorio_Tecnico_01_Analise_Geral.html
Objetivo: Consolidação e análise da pesquisa científica sobre ludopatia e BETs, apresentando dados epidemiológicos, referências bibliográficas, bases de dados consultadas e os três conectores temáticos (cultural, familiar e epidemiológico).

Conteúdo Principal:
  • Resumo executivo do problema de ludopatia no Brasil
  • Mapeamento dos 3 conectores temáticos com dados e evidências
  • Evolução epidemiológica: crescimento de 108 → 1.290 atendimentos SUS (2018-2023)
  • Documentação completa de bases de dados (DataSUS, Banco Central, CPI das BETs, academia)
  • Referências bibliográficas de artigos científicos 2024-2025
  • Legislação brasileira sobre apostas e saúde mental
  • Próximos passos metodológicos para análise exploratória de dados
Valor Didático: Demonstra capacidade de síntese de pesquisa multidisciplinar; articula dados públicos com literatura científica; propõe caminho metodológico claro.

💻 Relatório Técnico 02
Título: Análise Comentada do Script (wireframe2.html)
Arquivo: Relatorio_Tecnico_02_Analise_Script.html
Objetivo: Documentação técnica completa do código frontend (HTML, CSS, JavaScript) que implementa o portal web de apresentação do projeto.

Conteúdo Principal:
  • Stack tecnológico: HTML5, CSS3 (variáveis, Grid), JavaScript vanilla, Chart.js
  • Análise estrutural: head, navbar, componentes principais
  • Variáveis CSS customizadas (--primary, --accent, paleta de cores)
  • Implementação de gráficos: Line Chart (evolução SUS) e Radar Chart (vulnerabilidades)
  • Design responsivo: Grid auto-fit, breakpoints, mobile-first
  • Performance: CDN, inline CSS, SVG embutido
  • Acessibilidade (A11y): ARIA labels, semântica HTML, contraste
  • Segurança: CSP, XSS prevention, validação de URLs
  • Decisões de design justificadas; avaliação de qualidade com notas (7-9/10)
Valor Técnico: Demonstra compreensão de desenvolvimento web moderno; documentação profissional de código; recomendações de otimização; alinhamento com princípios de acessibilidade.
